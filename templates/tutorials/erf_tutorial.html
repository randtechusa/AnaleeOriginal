Explanation Recognition Feature (ERF)
How It Works:

The Explanation Recognition Feature (ERF) automatically finds transactions with similar descriptions in your records.

What It Does
  • Identifies transactions with 70%+ text similarity
  • Groups similar transactions together
  • Automatically suggests matching explanations
Benefits
  • Saves time on repetitive entries
  • Ensures consistent explanations
  • Reduces manual data entry errors
How to Use:
  1. Enter an explanation for any transaction
  2. Similar transactions will be identified automatically
  3. Click on a similar transaction to apply the same explanation
  4. Confirm to update all similar transactions at once
Pro Tip: ERF works best with detailed explanations that include specific keywords relevant to the transaction.

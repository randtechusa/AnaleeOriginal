Account Suggestion Feature (ASF)
How It Works:

The Account Suggestion Feature (ASF) intelligently recommends the most appropriate account for each transaction.

What It Does
  • Analyzes transaction descriptions
  • Uses historical account selections
  • Considers transaction patterns
Benefits
  • Speeds up transaction processing
  • Reduces categorization errors
  • Increases consistency in accounting
How to Use:
  1. View the transaction details and description
  2. ASF will automatically suggest the most appropriate account
  3. See the confidence level and reason for each suggestion
  4. Either accept the suggestion or select a different account
Pro Tip: ASF improves over time as you process more transactions, learning from your account selections to make better suggestions.

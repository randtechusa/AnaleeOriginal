Explanation Suggestion Feature (ESF)
How It Works:

The Explanation Suggestion Feature (ESF) automatically generates professional explanations for your transactions.

What It Does
  • Generates contextual explanations
  • Uses transaction description data
  • Learns from your existing explanations
Benefits
  • Saves time writing explanations
  • Ensures clear, professional wording
  • Creates consistent documentation
How to Use:
  1. Click on a transaction without an explanation
  2. ESF will automatically suggest an appropriate explanation
  3. Review and edit the suggestion if needed
  4. Save to apply the explanation to the transaction
Pro Tip: ESF improves with usage - the more explanations you create and edit, the better it becomes at suggesting relevant explanations for new transactions.
